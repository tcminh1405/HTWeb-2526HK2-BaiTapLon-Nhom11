# HTWeb-2526HK2-BaiTapLon-Nhom11

## 🏨 Vinpearl Luxury Nha Trangu - Luxury Experience

Final term project for Web System / Web Programming course (Academic Year 2025-2026, Semester 2).
Topic: Build a hotel introduction and online booking website.

**Team**: Group 11

### 📑 Overview
The **Vinpearl Luxury Nha Trangu** website is designed with a modern and luxurious style (Luxury Experience), providing an optimal user experience for exploring information and booking hotel rooms. The interface is user-friendly and fully responsive across all devices.

### ✨ Main Features & Pages
- **Home (`index.html` / `home.html`)**: Hotel overview, video banner, short introduction.
- **Rooms & Suites (`rooms.html` / `roomdetail.html`)**: List of room types, pricing, and detailed information.
- **Services (`services.html`)**: Spa, Gym, Conference, Pool, etc.
- **Dining (`dinning.html`)**: Restaurant info, buffet, and bar.
- **Gallery (`gallery.html`)**: Images of the resort space and hotel events.
- **Contact (`contact.html`)**: Contact form, map, and online support information.
- **Booking (`booking.html` / `payment.html`)**: Process to select dates, rooms, extra services, and secure payment.
- **News & FAQ (`news.html`, `faqs.html`)**: Updates on events, offers, and frequently asked questions.
- **User (`login.html`)**: Login/Register for customers.

### 🛠 Technologies Used
- **Frontend Core**: HTML5, CSS3, Vanilla JavaScript.
- **Framework & Libraries**: 
  - [Bootstrap 5.3.8](https://getbootstrap.com/) (Offline) for grid system and UI components (Navbar, Modal, Carousel, Cards, etc.).
  - [FontAwesome 7.2.0](https://fontawesome.com/) (Offline) for diverse icon system.
- **Fonts**: Elegant typography with Google Fonts (Playfair Display, Poppins).

### 🚀 Setup & Run Instructions
1. Clone this repository or download the source code as a ZIP file.
2. Extract the folder (if downloaded as ZIP).
3. Open the `index.html` file directly using any web browser (Google Chrome, Microsoft Edge, or Firefox recommended).
   - *(Tip: It is highly recommended to use the **Live Server** extension on VS Code for the best hot-reload experience).*
4. No need to install NodeJS or other backend services (this is a pure static frontend project).
